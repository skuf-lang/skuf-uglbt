# Contributing to uglbt

We welcome contributions! All contributions must be made via **fork and pull request**.

## How to contribute

1. **Fork** this repository
2. **Clone** your fork locally
3. **Create a branch** for your change
4. **Make your changes** and commit
5. **Push** to your fork
6. **Open a pull request** against `main`

```bash
# Fork via GitHub, then:
git clone https://github.com/<your-username>/skuf-uglbt.git
cd skuf-uglbt
git checkout -b my-feature
# make changes
git add -A
git commit -m "Add my feature"
git push origin my-feature
# Open PR on GitHub
```

## Guidelines

- Keep PRs focused — one feature or fix per PR
- Follow existing code style (Skuf conventions)
- Update the README if adding new API surface
- Add or update `.skf` stubs for any new functions

## Direct pushes

Direct pushes to `main` are **not allowed**. All changes go through pull requests with review.
